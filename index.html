Chatbot Widget Test Page

External HTML Test

This is a standalone HTML file that loads the chatbot widget from your SvelteKit application.

The widget should appear in the bottom-right corner of the page.

About this Test

This HTML file demonstrates how a customer would integrate your chatbot widget into their website.

The widget is loaded via a simple script tag that points to your SvelteKit application's embed.js endpoint.

For production use, you would host your SvelteKit app on a public domain and update the script src accordingly.

Features Demonstrated

  • Loading the widget from an external site
  • Using the JavaScript API to control the widget
  • Domain detection (the widget should detect it's running on a local file)
